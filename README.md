# Xen for backend
これはXenのバックエンドプロジェクトです.  
XenからのHTTPリクエストの受け口と警策グローブを動作させる役割を果たしています.

...予定でしたがスイッチング機構が上手く行かず頓挫しました.

### 動作環境

|||
|:--|:--|
|マシン|Raspberry Pi Zero WH|
|OS|Raspbian Stretch Lite(April 2019)|
|Webサーバ|nginx|
|PHP|PHP 7.0|
|Python|Python 3.5.3|
